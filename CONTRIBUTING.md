# Contributing

Thanks for considering contributing to SignalFlow. High-level guidelines:

- Open issues for bugs or feature requests.
- Fork the repo and create topic branches for changes.
- Keep changes small and focused; add tests for new behavior.
- Run `python -m pip install -r requirements.txt` and `pytest` before submitting PRs.
- Follow the security guidelines in `SECURITY.md` — do not add functionality that harvests secrets or bypasses platform protections.
